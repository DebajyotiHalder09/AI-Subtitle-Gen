Attention

Ensure that you are on the home page.

You can navigate to the home page by clicking on the TITLE button.

First Step

-> Select the MP3 file from your device.

-> Click the "Open" button at the bottom-right of the popup window.

Second Step

-> If you have a JSON file corresponding to the selected audio file, choose and open it.

-> If you do not have a JSON file, proceed to the third step.

Third Step

-> Ensure that your audio file is selected.

-> Click the "Generate" button to create the subtitle file in JSON format.

-> Wait for the subtitle file to be generated.

-> Once the file is generated, download and save it with an appropriate name.

-> Choose the generated JSON file from the second step to continue.

Play and Enjoy
